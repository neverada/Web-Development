To achieve this, we need to use: 1. **Bootstrap Grid (`row` and `col`)**: To divide the screen space. 2. **Responsiveness Classes**: To show the sidebar on desktop (`d-lg-block`) and hide it on mobile (`d-none`). 3. **Bootstrap Offcanvas**: To make the sidebar slide out on mobile. CSS (Styling the Dark Sidebar) Add this to your stylesheet (`style.css`): ```css /* Custom Mantu Sidebar Styling */ .category-sidebar { background-color: #1a222f; /* Dark background color */ color: #fff; min-height: 100%; /* Ensures it fills vertical space */ } /* Category Item Styling */ .category-item { display: flex; justify-content: space-between; align-items: center; padding: 1rem 1.5rem; cursor: pointer; font-size: 1rem; font-weight: 500; border-bottom: 1px solid rgba(255, 255, 255, 0.1); transition: background-color 0.2s, color 0.2s; } .category-item:hover, .category-item.active { background-color: #2b3342; /* Slightly lighter background on hover/active */ color: #fff; } /* Submenu/Collapse styling (for nested items like T-shirts) */ .category-submenu { list-style: none; padding: 10px 0 10px 20px; /* Indent sub-items */ background-color: #1a222f; /* Maintain dark background */ } .category-submenu li a { color: #b0b4ba; /* Lighter text for sub-items */ display: block; padding: 8px 0; text-decoration: none; } .category-submenu li a:hover { color: #3d57ff; /* Primary color on hover */ } ``` ### 2. We can use a tiny bit of JS to achieve this for better UX. ```javascript document.addEventListener('DOMContentLoaded', function () { // Select all items that toggle a collapse panel document.querySelectorAll('.category-item').forEach(item => { item.addEventListener('click', function() { const targetId = this.getAttribute('data-bs-target'); const targetElement = document.querySelector(targetId); const icon = this.querySelector('.fa-chevron-down'); if (targetElement && icon) { // Wait briefly for Bootstrap's collapse transition to check state setTimeout(() => { if (targetElement.classList.contains('show')) { // Panel is open, rotate icon up icon.style.transform = 'rotate(180deg)'; } else { // Panel is closed, rotate icon down icon.style.transform = 'rotate(0deg)'; } }, 50); // Small delay } }); }); }); ``` ### Summary of Responsiveness: | Screen Size | Classes Applied | Result | | :--- | :--- | :--- | | **XS, SM, MD** (< 992px) | `d-lg-none` hides desktop sidebar. | The **"Shop Filters" button** appears and opens the **Mobile Offcanvas** (Image 3). | | **LG, XL, XXL** (≥ 992px) | `d-none d-lg-block` shows desktop sidebar. | The sidebar is fixed on the left in the `col-lg-3` grid space (Image 2). | This setup provides the required dual functionality and visual styling.
